class chemopy.orthogonal_corrections.OrthogonalProjection(P)[source]

Bases: BaseEstimator, TransformerMixin

Orthogonal Projection Transformer.

This class implements a transformer that performs orthogonal projection on input data.

Parameters:

Parray-like, shape (n_features, n_features)

The orthogonal matrix used for projection. It should be symmetric and orthogonal, i.e., P @ P.T = P.T @ P = I, where I is the identity matrix.

Methods:

Fits the transformer to the data. This method does nothing and only exists to comply with the scikit-learn TransformerMixin interface.
